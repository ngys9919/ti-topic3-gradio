# 👗 Fashion-MNIST ResNet Model Explorer

Train a custom Mini-ResNet on the Fashion-MNIST dataset and explore how different hyperparameters affect performance.

## Features
- Custom Mini-ResNet architecture with configurable residual blocks
- Multiple optimizer choices: Adam, SGD, SGD+Momentum, RMSprop, AdamW
- Adjustable learning rate, batch size, and number of epochs
- Optional early stopping
- Training curves and confusion matrix visualization
